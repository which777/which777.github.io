HTML学习笔记

1. 初识HTML

HTML的基本概况

HTML 负责网页的结构

css 负责网页的样式

JavaScript 负责网页交互

HTML是一种建立网页文件的语言，通过标记式的指令 (Tag)，将文字，图形、动画、声音、表格、链接、影像等内容显示出来。 事实上，每一个HTML文档都是一种静态的网页文件，这个文件里面包含了HTML 指令代码 ，这些指令代码并不是一种 程序语言 ，只是一种排版网页中资料显示位置的标记结构语言，易学易懂，非常简单。

HTML5基本信息

在HTML中标签被分为双标签和单标签
双标签： 例如 body,body
单标签（空标签）： 例如 img,hr

HTML基本骨架

DOCTYPE标签:是告诉浏览器，我们要使用什么规范
html标签：表示该文件是以超文本标记语言（HTML）编写的
head标签：代表网页的头部
meta标签：描述形标签，它用来描述我们网站的一些信息
title标签: 代表网页的标题
body标签: 代表网页的主体内容。

4.链接标签

href:必填，表示要跳转到哪个页面

_self :会在相同的框架或者窗口中打开链接。

_blank:在新标签中打开

_parent:会在父窗口或者包含来超链接引用的框架的框架集中打开链接

_top:会清除所有被包含的框架，并打开链接
